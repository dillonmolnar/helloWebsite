Create a repo Open Amplify

What’s here

  • index.html — this page
  • styles.css — layout and theme

No build step required. Amplify can serve these files as-is from the repository root.

Quick deploy checklist

  1. Initialize git in this folder and push to GitHub.
  2. In AWS Amplify: Host web app → connect GitHub.
  3. Use the default build settings for a static site (or add a minimal amplify.yml if prompted).
  4. Save and deploy; your URL appears when the first build finishes.
